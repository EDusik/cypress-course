<div id="tecnologies"> 

### :rocket: Technologies
This project uses these technologies:
- [React](https://reactjs.org/)
- [Cypress](https://www.cypress.io/)
- [Mirage](https://miragejs.com/)
</div>

### :computer: Dependencies
<div id="dependencies"> 

#### Start React Project
``` npm
npx create-react-app cypress-course
```
#### Install Cypress
``` npm 
npm install cypress
```
<img src="./cypress/images/installing-global.gif" width="50%"> 

##### Finish Installing Cypress
You can follow the step by step through this 
<a href="https://docs.cypress.io/guides/getting-started/installing-cypress.html#yarn-add">tutorial</a>

#### Install Mirage
``` npm 
npm install --save-dev miragejs
```
</div>

### :zap: Commands
<div id="commands">

#### Run the project
``` npm 
npm start
```
Open <a href="http://localhost:3000/">http://localhost:3000/</a>

##### Run Cypress


``` npx 
npx cypress open
```
</div>
